# SSRF Playground

Интерактивное обучающее приложение для изучения уязвимости Server-Side Request Forgery (SSRF).

## Что внутри

- **Теоретическая часть** — объяснение SSRF «на пальцах» с аналогией про секретаря, схемой атаки и ссылками на OWASP/PortSwigger.
- **Интерактивная песочница** — 5 сценариев SSRF-атак с эмуляцией ответов сервера:
  1. Чтение облачных метаданных AWS
  2. Доступ к внутренней админ-панели (localhost)
  3. Чтение локальных файлов (file://)
  4. Сканирование внутренних портов
  5. Обход через DNS Rebinding
- **Переключатель темы** — светлая / тёмная (сохраняется в localStorage).
- **Адаптивная вёрстка** — mobile (320px+), tablet (768px+), desktop (1200px+).
- **Кнопка «Наверх»** — появляется при прокрутке > 300px.

## Стек

- React 19 + Vite
- CSS Custom Properties (темы)
- React Icons
- react-syntax-highlighter (подсветка JSON)

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
npm run preview
```

## Деплой на Railway

1. Репозиторий подключён к Railway.
2. Railway автоматически определяет статический сайт через `static.json`.
3. При каждом push в `main` происходит автоматический деплой.
